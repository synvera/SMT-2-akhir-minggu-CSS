Nama Website

KasKelasKu

Tujuan Website

Website ini dibuat untuk membantu pengelolaan kas kelas supaya lebih rapi dan transparan. Bendahara bisa mencatat pemasukan dan pengeluaran, sementara siswa bisa melihat laporan kas tanpa harus tanya langsung.

Struktur Halaman

Home = berisi tampilan utama dan penjelasan singkat website

Login/Register = untuk masuk ke akun

Dashboard = menampilkan ringkasan kas

Transaksi = input pemasukan dan pengeluaran

Laporan = melihat data kas secara detail

Profile = data pengguna

Footer = informasi tambahan dan kontak

Fitur Website

Login dan register user

Input data kas (pemasukan & pengeluaran)

Menampilkan total saldo kas

Riwayat transaksi

Laporan kas yang bisa dilihat semua anggota

Tampilan responsive (bisa di HP dan laptop)

Desain Warna

Warna utama: Hitam (#000000)

Warna sekunder: Putih (#ffffff)

Aksen: Abu-abu (#888888)

Konsep warna dibuat monochrome supaya terlihat simpel, modern, dan nyaman dilihat
